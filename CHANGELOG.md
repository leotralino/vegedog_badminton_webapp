# Changelog

## [0.9.3] - 2026-04-17
- PWA 安装支持：设置页标题旁新增安装按钮，iOS 显示步骤指引弹窗，Android 触发原生安装提示
- 已安装时自动隐藏安装按钮

## [0.9.2] - 2026-04-17
- 暖色调 UI 主题：背景、卡片、边框、输入框、顶底导航栏统一采用暖米色调
- 顶部导航栏图标改为 Lucide 房屋风格
- 登录后强制设置接龙昵称：未设昵称用户自动跳转设置页，中间件全路径拦截
- 设置页：昵称字段改为"接龙昵称（必填）"，新增设置密码（可选）功能
- 设置页：退出登录改用应用内弹窗确认
- 统计：改为 3 列布局，调整顺序，新增菜狗杯参与（待接入）
- 修复：设置页 upsert 不再覆盖已有头像
- 修复：中间件昵称检查排除 /api/ 路由

## [0.9.1] - 2026-04-16
- 手绘狗狗吉祥物水印：各标签页配对专属图案（接龙/历史/设置/菜狗杯）
- 修复吉祥物水印被卡片遮挡的层叠顺序问题（CSS z-index）
- 顶部导航栏"菜狗主页"图标改为 SVG 房屋图标
- 设置页取消/保存按钮顺序调整（取消在左，保存在右）

## [0.9.0] - 2026-04-15
- 参与记录热力图（5个月滚动，GitHub 风格）
- 统计：参与次数按场次去重，新增帮助+1人数统计
- 加时次数改为按场次去重
- 迟退标签改为中文"迟退 ⚠️"
- 设置账户页改为分项编辑（每个字段单独编辑/保存/取消）
- 昵称编辑增加特殊字符与 emoji 校验
- 退出登录增加确认弹窗
- 数据库：所有增量 patches 合并入 schema.sql，新项目只需一个文件

## [0.8.0] - 2026-04-15
- 关注系统：关注好友后对方发起新接龙时自动发送邮件通知
- 设置页新增统计标签页（接龙次数、候补次数、发起次数、加时次数）
- 设置页新增关注标签页（关注/取消关注用户，查看粉丝数）
- Vercel Analytics 集成

## [0.7.0] - 2026-04-15
- 多管理员支持：接龙发起人可添加副管理员，副管理员拥有同等操作权限
- 新建接龙时可直接指定副管理员
- 发给球馆功能：管理员可将接龙信息格式化后发送至球馆邮箱
- 支持通过环境变量配置多个球馆邮箱（逗号分隔）
- 添加 PWA 应用图标（180/192/512px）

## [0.6.0] - 2026-04-14
- Venmo 深链接：优先唤起手机 App，不可用时回退至网页版
- 点击 Venmo 按钮前弹出提醒对话框（强调需手动确认付款）
- 付款方式支持每位收款人单独填写金额（$X/人）
- 管理员可编辑/删除已添加的收款人
- 所有付款完成后显示庆祝横幅
- 锁定接龙中管理员可通过搜索框快速定位参与者
- 修复 payment_methods 表缺少 UPDATE/DELETE RLS 策略的问题
- 移除付款记录中 $0.00 占位符显示

## [0.5.0] - 2026-04-13
- 接龙锁定时自动通过数据库触发器批量生成付款记录
- 管理员可逐一标记参与者付款状态（已付/未付/豁免）
- 参与者可自行标记自己的付款状态
- 付款记录实时同步（Supabase Realtime）
- 付款区域支持用户搜索下拉框添加收款人
- 收款人 Venmo 账号自动从个人资料读取

## [0.4.0] - 2026-04-13
- 管理员可将锁定接龙移入历史（closed 状态）
- 已关闭接龙全部内容变为只读
- 新增加时（+时）标记功能，管理员可标记哪些参与者留下加时
- 加时名单独立展示面板
- 发起接龙按钮从导航栏移至接龙列表页标题旁
- 离开新建接龙页时弹窗确认，防止误操作丢失填写内容
- 未登录用户访问任何页面自动跳转登录页

## [0.3.0] - 2026-04-13
- 全中文界面
- 底部导航栏：接龙 / 历史 / 设置
- 顶部导航栏"菜狗主页"入口
- 骨架屏加载动画，减少白屏等待
- 地点支持预设选项下拉 + 自定义地点双字段（名称+地址）
- 时间选择器限制为 15 分钟粒度
- 用系统内弹窗替换浏览器原生 confirm/alert
- 标题旁添加 iOS 风格分享按钮
- 邮件/密码注册用户自动生成 DiceBear 头像

## [0.2.0] - 2026-04-13
- 邮件密码登录
- 魔法链接（无密码邮件登录）
- 密码重置流程（邮件回调 + 设置新密码页）
- 支持 +1 报名：同一账号可为多人报名（不同名字视为不同参与者）
- 乐观更新：加入/退出操作立即反映在 UI，无需等待服务器
- 修复时区错误导致的服务端/客户端渲染不一致问题
- 修复 TypeScript 类型错误，移除 Database 泛型

## [0.1.0] - 2026-04-12
- 项目初始化（Next.js 15 + Supabase + Tailwind CSS）
- 创建接龙：标题、地点、时间、人数上限、场地数、费用、截止退出时间
- 加入接龙 / 退出接龙（截止前免罚，截止后标记迟退）
- 候补队列：超过人数上限自动进入候补，有人退出后自动递补
- 并发安全：使用 pg_advisory_xact_lock 防止超员竞争
- Google OAuth 登录
- 全表 RLS 行级安全策略
- 新用户注册自动创建 profile（触发器）
